Pages, blog posts, products and events can be assigned to these categories.

### Input fields

<kbd>Title</kbd> Category title

<kb>Priority</kbd> The priority (and the language) determines the sorting of the categories.

<kbd>Thumbnail</kbd> An image for this category can be selected here.

<kbd>Language</kbd> The language of the category

<kbd>Description</kbd> The description of the category
