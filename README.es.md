# SwiftyEdit CMS

SwiftyEdit es un sistema de gestión de contenidos (CMS) ligero y de código abierto creado con 
PHP, MySQL o SQLite para la creación de sitios web modernos. 
Su núcleo minimalista está diseñado para ser simple pero altamente ampliable a través de complementos y temas, 
lo que lo hace adecuado tanto para proyectos pequeños como para soluciones web más grandes.

## Características

SwiftyEdit tiene una estructura muy simple y puede utilizarse tanto para los proyectos más pequeños como para desafíos más grandes. 
Los siguientes módulos están integrados en el sistema:

* El tipo de __gestión de páginas__ que debe tener un CMS.
* __Snippets__ (Fragmentos) - No escribas lo mismo una y otra vez.
* __Blog__ - Publica tus noticias, fotos, galerías o descargas y vídeos. Todo lo que quieras.
* __Tienda__ - Presenta tus ofertas. Vende tus cosas, incluidos productos digitales.
* __Eventos__ - Publica tus eventos y gestiona reservas y listas de invitados.

#### Requisitos técnicos

+ Software: PHP 8.3 o superior
+ Servidor Web: Apache con módulo PDO/SQLite y mod_rewrite
+ Base de datos: MySQL 5.6+ o SQLite

#### Descarga

Obtén la última versión en https://swiftyedit.org/download/

#### O instala usando Composer
```
composer create-project swiftyedit/swiftyedit
```

__Ten en cuenta:__
A partir de la versión 2.0, SwiftyEdit requiere el directorio /public/ como raíz del dominio. 
Por lo tanto, las actualizaciones automáticas de 1.x a 2.x no son posibles. Después de migrar a 2.x manualmente, 
las actualizaciones futuras podrán volver a instalarse automáticamente.
