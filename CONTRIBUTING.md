# Contributing to SwiftyEdit

Thanks for your interest in contributing!
Please note that all contributors must follow our [Code of Conduct](./CODE_OF_CONDUCT.md).

## Getting Started
- Fork the repo and clone your fork
- Install latest dev version:
  ```bash
  composer create-project swiftyedit/swiftyedit=dev-main swiftyedit-dev --stability=dev
  cd swiftyedit-dev
  composer install  # Dependencies + dev tools

## Development Workflow

 - Create branches: feature/short-description or fix/issue-number
 - Keep PRs small and focused
 - Reference related issues in PR description
 - Update README/docs for user-facing changes
