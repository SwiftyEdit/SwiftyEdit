# Benutzerverwaltung

Es gibt zwei Möglichkeiten, Benutzerkonten zu erstellen.
Über das Frontend können sich Nutzer selbstständig registrieren
(sofern diese Funktion in den Einstellungen aktiviert ist), oder das Benutzerkonto wird über das Backend erstellt.

Wenn sich ein Benutzer über das Frontend registriert,
muss er sein Konto bestätigen. D.h. er muss auf den → klicken,
der automatisch an seine E-Mail-Adresse gesendet wird.

## Die (Benutzer-)Rechteverwaltung

SwiftyEdit verfügt über eine effektive Rechteverwaltung für Besucher, Moderatoren und Administratoren.

Jeder Benutzer oder Administrator kann mit unterschiedlichen Benutzerrechten ausgestattet werden.
Die einzelnen Optionen findest du unter ▶ Backend ➜ Benutzer ➜ **Bearbeiten** im Tab Passwort- und Rechteverwaltung.
