# Features

Features are listed in the frontend. The default theme shows these details in table form.
The left column shows the title and the right column shows the text.

When a new product is created, the feature can simply be adopted.
However, in each product you can override the feature created here (that is, the text).

An example

| Title   | Text   |
|---------|--------|
| Content | 0.33 l |
