# Orders

## Payments

If an order is not automatically marked as paid by a payment plugin
(for example bank transfer), you can mark the orders accordingly here once the payment has been received.

Digital products can only be downloaded once the order has been marked as paid.
The buyer can then download the file on their profile page.

## Order status

 - Received
 - Completed
 - Cancelled
